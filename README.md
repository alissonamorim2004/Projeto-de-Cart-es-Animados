# Projeto de Cartões Animados

Este projeto é uma interface de cartões animados criada com **HTML** e **CSS**, ideal para exibir informações de forma visualmente atraente e interativa.

## Demonstração

A interface consiste em três cartões principais, cada um com uma imagem e uma breve descrição. Os cartões expandem ao passar o mouse, revelando mais detalhes e um botão de "Saiba mais". Eles são responsivos e se adaptam bem a dispositivos móveis.

## Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **Google Fonts**: Poppins
- **Imagens**: Usadas como exemplo

## Funcionalidades

- Efeitos de hover nos cartões, que aumentam a altura e revelam mais conteúdo.
- Estilo responsivo para dispositivos móveis, ajustando o tamanho dos cartões.
- Animações suaves nos elementos visuais (imagens e textos).
